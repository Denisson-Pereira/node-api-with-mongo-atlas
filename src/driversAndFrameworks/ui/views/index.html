/products

GET
POST
PUT
DELETE

Esta rota lida com produtos. Você pode usar os seguintes métodos:

  • GET: Retorna todos os produtos.
  • POST: Cria um novo produto.
  • PUT: Atualiza um produto existente.
  • DELETE: Deleta um produto.

Atributos de um Produto:

  • _id: Identificador único do produto (tipo string, gerado automaticamente pelo MongoDB).
  • name: Nome do produto (tipo string).
  • description: Descrição do produto (tipo string).
  • price: Preço do produto (tipo string, pode ter valores numéricos formatados como string).
  • evaluation: Avaliação do produto (tipo string, geralmente com uma pontuação numérica representada como string).
  • image: URL da imagem do produto (tipo string).
  • createdAt: Data de criação do produto (tipo string, formato ISO 8601).
  • updatedAt: Data da última atualização do produto (tipo string, formato ISO 8601).
  • __v: Versão do documento (campo interno do Mongoose, tipo inteiro, controlado automaticamente pelo banco de dados).

Exemplo de um Produto:


{
    "_id": "676f18936a4e2bd8fea621cb",
    "name": "Produto Teste",
    "description": "Descrição do produto de teste",
    "price": "999",
    "evaluation": "4.5",
    "image": "https://exemplo.com/imagem.jpg",
    "createdAt": "2024-12-27T21:13:55.167Z",
    "updatedAt": "2024-12-28T02:08:14.875Z",
    "__v": 0
}

Esta rota lida com categorias. Você pode usar os seguintes métodos:

  • GET: Retorna todas as categorias.
  • POST: Cria uma nova categoria.
  • PUT: Atualiza uma categoria existente.

  • createdAt: Data de criação da categoria (tipo string, formato ISO 8601).
  • updatedAt: Data da última atualização da categoria (tipo string, formato ISO 8601).
