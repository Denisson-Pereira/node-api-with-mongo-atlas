API Documentation

/products

GET
POST
PUT
DELETE

Esta rota lida com produtos. Você pode usar os seguintes métodos:

  • GET: Retorna todos os produtos.
  • POST: Cria um novo produto.
  • PUT: Atualiza um produto existente.
  • DELETE: Deleta um produto.

Estrutura de um Produto:


export interface IProduct {
    name: string;
    description: string;
    price: string;
    evaluation: string;
    image: string;
}
